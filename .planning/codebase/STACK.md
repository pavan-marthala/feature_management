# Tech Stack
Date: 2026-04-18

## Overview
A high-performance, reactive Feature Management platform designed for modern microservices and front-end applications.

## Backend Stack
- **Language**: Java 25
- **Framework**: Spring Boot 4.0.5
- **Runtime**: Project Reactor (WebFlux)
- **Database Access**: Spring Data R2DBC
- **Code Generation**: Lombok, MapStruct, OpenAPI Generator
- **Build Tool**: Maven

## Frontend Stack
- **Framework**: Vue.js 3.5.30 (Composition API)
- **Language**: TypeScript 5.9.3
- **Build Tool**: Vite 7.3.1
- **Styling**: Tailwind CSS 4.2.2
- **State Management**: Pinia 3.0.4
- **Routing**: Vue Router 5.0.3
- **Linting/Formatting**: oxlint, eslint, oxfmt

## Database
- **Primary Datastore**: PostgreSQL
- **Schema Management**: Liquibase

## Infrastructure
- **Containerization**: Docker
- **Orchestration**: Docker Compose
- **Proxy/Gateway**: Nginx
